# Задание 1. Решить задачу на построение нового дерева по заданному (map)

## Задача 8

### Текст задачи

Дерево содержит вещественные числа. Округлить каждый элемент.

### Алгоритм решения

Входные данные:

depth (целое число) — глубина дерева.

originalTree — бинарное дерево, заполненное случайными вещественными числами типа float.

Выходные данные:

roundedTree (структура данных) — новое бинарное дерево той же структуры, но с округленными значениями в узлах.

Логика работы:

Определение структуры (Tree):

Создан тип данных.

Тип данных имеет два состояния: Empty (пустой узел/лист) и Node (узел, содержащий значение и ссылки на два дочерних дерева).

Генерация дерева (generateRandomTree):

Используется рекурсия до достижения заданной глубины depth.

Для каждого узла генерируется вещественное число от 0 до 100 с помощью rnd.NextDouble() * 100.0.

Базовый случай: При достижении нулевой глубины возвращается Empty, что останавливает рекурсию.

Трансформация через Map (treeMap):

Используется функция, реализующая Map для деревьев.

Обход: Программа посещает каждый узел и применяет к его значению переданную функцию f.

Вместо изменения дерева, конструктор Node на каждом шаге рекурсии собирает новый узел, формируя новое дерево в памяти.

Вывод дерева (printTree):

центрированный обход - вывод в виде (Право -> Узел -> Лево).

Используется передача параметра indent (отступ), который увеличивается на каждом уровне рекурсии. Это позволяет корректно отобразить дерева в текстовой консоли.

Завершение:

Программа последовательно выводит исходное дерево с дробными числами и результирующее дерево с округленными значениями.

# Задание 2. Решить задачу на получение заданного результата для указанного дерева (fold)

## Задача 8

### Текст задачи

Сколько раз входит заданный элемент в дерево?

### Алгоритм решения

Входные данные:

depth (целое число) — глубина дерева.

target (целое число) — значение, количество вхождений которого необходимо подсчитать (вводится пользователем).

tree (структура данных) — бинарное дерево, заполненное случайными целыми числами.

Выходные данные:

count (целое число) — итоговое количество узлов дерева, значение которых совпадает с target.

Логика работы:

Определение структуры (Tree):

Создан тип данных.

Тип данных имеет два состояния: Empty (пустой узел/лист) и Node (узел, содержащий значение и ссылки на два дочерних дерева).

Генерация дерева (generateRandomTree):

Используется рекурсия до достижения заданной глубины depth.

Для увеличения вероятности повторений в узлах генерируются числа в диапазоне от 1 до 5 с помощью rnd.Next(1, 6).

Свёртка дерева (treeFold):

Применяется функция fold для сбора древовидной структуры в единое значение.

Функция принимает начальное значение счетчика (acc = 0).

Обход: Программа сначала собирает данные из левой ветви, затем применяет логику к текущему узлу, и передает полученный результат в правую ветвь.

Подсчет вхождений:

В качестве функции обработки передается: (fun acc x -> if x = target then acc + 1 else acc).

Если значение узла x совпадает с искомым target, acc увеличивается на 1. В противном случае значение acc проносится дальше без изменений.

Завершение:

После полного обхода дерева итоговая сумма выводится на экран. Вывод деерва с помощью функции printTree.
